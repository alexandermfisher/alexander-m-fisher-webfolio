Webfolio
Description

This project involves the development of a custom database server that offers users a simplified version of the SQL database query language. The server is built from scratch using Java, Maven, and JUnit, ensuring robust functionality and thorough testing. The core components, including the parser and interpreter, are implemented from the ground up. This project not only demonstrates advanced programming skills but also provides a deeper understanding of database internals and query processing.

Features
  • Implements specified BNF
  • Utilizes local filesystem for storage using tab files
  • Includes server and client scripts for communication
Tools and Languages Used
  • IntelliJ IDEA for development
  • Java programming language
  • Maven for project management and build automation
  • JSON for database metadata
  • JUnit for testing
  • GitHub for version control
Key Concepts and Learning
  • Implemented a custom parser to generate a parse tree
  • Utilized the Visitor design pattern to implement the interpreter
  • Employed composite data structures, for efficient data storage and retrieval
  • Applied the Singleton design pattern to ensure a single instance of critical server components
  • Learned and implemented robust exception handling strategies

Description

Stag is an engaging general-purpose socket-server game-engine for text adventure games, reminiscent of classics like Zork. It features a robust game engine developed entirely from scratch using Java, Maven, and JUnit for testing. The game entities and actions are read and parsed from dot and xml configuration files.

  • Flexible Gameplay Configurations
  • Basic Command Language

  • Dynamic World Environment
  • Multiple Client Connections
